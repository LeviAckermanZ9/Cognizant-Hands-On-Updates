# Data Structures and Algorithms

## Arrays & Strings
- **Time Complexity**: Access O(1), Search O(N), Insertion/Deletion O(N).
- Key algorithms: Two pointers, Sliding window, Prefix sum.

## Linked Lists
- Types: Singly, Doubly, Circular.
- Advantages: Dynamic size, ease of insertion/deletion.
- Disadvantages: Sequential access.

## Trees & Graphs
- **Binary Search Tree (BST)**: Left child < Node < Right child. Average search O(log N).
- **Graph Traversals**: Breadth-First Search (BFS) and Depth-First Search (DFS).
- **Shortest Path**: Dijkstra's algorithm.

## Sorting Algorithms
- **Quick Sort**: O(N log N) average case. Divide and conquer.
- **Merge Sort**: O(N log N) worst case. Stable sort.
