# Week 1: Foundational Engineering Skills

This week covers the core foundational skills required for the Java Full Stack Engineering track.

## Topics Covered
- **Design Patterns and Principles**: SOLID principles, Creational, Structural, and Behavioral patterns.
- **Data Structures and Algorithms**: Arrays, Linked Lists, Trees, Graphs, Sorting, and Searching.
- **PL/SQL Programming**: Database programming, stored procedures, triggers, and cursors.
- **TDD & Logging**: Test-Driven Development with JUnit, and logging frameworks (Log4j/SLF4J).
